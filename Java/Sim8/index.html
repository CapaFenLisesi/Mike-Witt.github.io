Sim8 Download Page

Sim8 is a simulation of a very simple 8 bit microcomuter. This program is provided for free, for instructional use. Sim8 is written in Java, and you must have a fairly current Java Runtime Environment installed on your system in order to run it. For information on obtaining Java go to: http://java.sun.com
