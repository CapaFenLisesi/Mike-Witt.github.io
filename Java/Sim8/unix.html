Download Sim8 for a Unix system

  1. To start the download: click here

  2. After the download completes, make the directory you want to put Sim8 in, and then extract it from the tar file. After that, you can either read the documentation by pointing a browser at the file Documentation/index.html, or else start Sim8 as shown below and then use the Information->Documentation menu. 
    	mkdir Sim8
	cd Sim8
	tar xvf Sim8-13Feb2002.tar
	java -jar Sim8.jar
