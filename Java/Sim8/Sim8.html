Sim8 Microcomputer Simulator

Note: Make sure you have the Java 2 Runtime Environment, Standard Edition, version 1.3.0, which includes the Java Plugin 1.3. This plugin is needed to run the Sim8 applet. It can be downloaded from http://java.sun.com/products/plugin/index.html.

If you do not have the plugin you will probably get one of these two errors:
Security violation: method verification error
java.lang.NoClassDefFoundError

You do not need to run the MicroComputer Simulator as an Applet. You can download the program and run it as a Java Application on your system. To download the Simulator click here.
